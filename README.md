## Overview

<img width="1388" height="863" alt="image" src="https://github.com/user-attachments/assets/fc0c7a40-2d0c-4ccc-8c88-a2a8649aebd0" />


This project is a responsive article listing page based on the provided design. It displays a list of technology-related articles with an image, title, and publication date, focusing on clean spacing and typography.

### What I learned

- How to build a clean layout using **Flexbox** for aligning images and text content.
- How to keep typography consistent using a custom font and sensible spacing.
- How to make the layout responsive by adjusting spacing and sizing for smaller screens.

### Useful resources

- [A Complete Guide to Flexbox (CSS-Tricks)](https://css-tricks.com/snippets/css/a-guide-to-flexbox/) - Helped me understand flex alignment, spacing, and common layout patterns.

### Built with

- Semantic HTML5 markup
- CSS (vanilla)
- Flexbox

## Features

- Responsive layout for desktop and mobile
- Each article includes an image, title, and publication date
- Clean spacing and typography matching the design
